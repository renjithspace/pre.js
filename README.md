# Pre.js
Pure JavaScript plugin for preserves spaces and line breaks


### Installation
**Bower**
```sh
bower install pre.js --save
```
**Direct download** from [latest releases](https://github.com/renjithvk/pre.js/releases)


### Usage
**Include**
```js
<script src="src/js/pre.js"></script>
```
**Initialize**
```js
new Pre({
    el: '.description',
    spaces: false,
    newline: true
});
```
